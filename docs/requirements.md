# قوانین نهایی سامانه بیان برتر

- نقش‌ها فقط `ADMIN`، `TEACHER` و `STUDENT` هستند و فقط یک مدیر وجود دارد.
- ورود همه با شماره تلفن و رمز عبور است. کد ملی رمز اولیه است و هر کاربر می‌تواند رمز را تغییر دهد.
- حساب دانش‌آموز حساب مشترک او و والدین محسوب می‌شود.
- فقط مدیر استاد و دانش‌آموز ایجاد می‌کند و تمام مدیریت داخل اپ انجام می‌شود.
- هر دانش‌آموز در هر لحظه حداکثر یک کلاس فعال دارد؛ تاریخچه عضویت حذف نمی‌شود.
- هر کلاس یک استاد دارد و شامل نام، روزهای هفته، ساعت شروع/پایان و تعداد جلسات است. ظرفیت نیاز نیست.
- پایان ترم کلاس را `COMPLETED` می‌کند و هیچ سابقه‌ای را حذف نمی‌کند.
- حضورغیاب پس از ثبت نهایی قابل ویرایش یا حذف نیست. برای تأخیر، تعداد دقیقه اجباری است.
- غیبت یا تأخیر یک اعلان اختصاصی برای حساب دانش‌آموز ایجاد می‌کند.
- مدیر برای همه دانش‌آموزان، همه استادان یا یک کلاس اعلان می‌فرستد؛ استاد فقط برای کلاس‌های خودش.
- بخش پیوست اعلان حفظ می‌شود و در مرحله API تکمیل خواهد شد.
- معیارهای نمره پویا هستند، مجموع بارم ۱۰۰ است و فقط مدیر کارنامه را ثبت و منتشر می‌کند.
- همه کارنامه‌های منتشرشده و سوابق ترم‌های گذشته نگهداری می‌شوند.
